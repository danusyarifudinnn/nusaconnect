# NusaConnect — Peta Interaktif Nusantara

NusaConnect adalah website peta interaktif yang memungkinkan pengguna menjelajahi provinsi di Indonesia melalui visual SVG yang dapat di-hover, di-klik, dan di-zoom untuk menampilkan informasi detail setiap daerah.

Proyek ini dibuat sebagai eksplorasi pembelajaran web interaktif menggunakan HTML, TailwindCSS, JavaScript (jQuery), serta manipulasi SVG untuk pengalaman pengguna yang imersif.

---

## 🎯 Fitur Utama

- Peta Indonesia berbasis SVG interaktif
- Efek hover provinsi (highlight & pop-up info)
- Animasi zoom ke provinsi saat diklik
- Panel informasi dinamis
- Routing dinamis ke halaman detail provinsi menggunakan parameter URL (`?region=`)
- Desain modern dengan TailwindCSS
